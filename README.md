# tampermonkey-scripts

Meine Tampermonkey-Userscripts.

## Installation

1. [Tampermonkey](https://www.tampermonkey.net/) als Browser-Erweiterung installieren (Chrome, Firefox, Edge, …)
2. In diesem Repo das gewünschte Script (`.user.js`-Datei) öffnen und auf **Raw** klicken
3. Tampermonkey erkennt die Datei automatisch und bietet die Installation an — **Installieren** klicken, fertig

Updates: Wenn im Script-Header eine `@updateURL`/`@downloadURL` auf die Raw-Datei zeigt, aktualisiert Tampermonkey das Script automatisch.
